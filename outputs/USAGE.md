## How to Run

### Method 1: Simple HTTP Server (Python)

```bash
cd outputs
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Method 2: Using Node.js http-server

```bash
cd outputs
npx http-server -p 8000
```

### Method 3: Any Other HTTP Server

Just serve the `outputs` directory with any web server.

## Customization

To add more papers, edit `data/papers.json`:

```json
{
  "id": "2501.00009",
  "title": "Your Paper Title",
  "authors": ["Author 1", "Author 2"],
  "affiliations": ["Institution"],
  "abstract": "Paper abstract...",
  "category": "cs.XX",
  "submitted": "2025-01-15",
  "pdf_url": "https://arxiv.org/pdf/2501.00009.pdf",
  "arxiv_url": "https://arxiv.org/abs/2501.00009"
}
```
